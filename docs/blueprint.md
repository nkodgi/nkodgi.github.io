# **App Name**: Priyanshu Mehta Portfolio

## Core Features:

- About Me: Display an 'About Me' section with a brief introduction and background information.
- Project Portfolio: Showcase projects with descriptions and links to repositories/live demos.
- Contact Form: Include a contact form for visitors to send messages.
- Resume Download: Allow visitors to download a resume in PDF format.
- Intelligent Category Generation: Use a tool that allows users to submit any external project description and an LLM determines the project catgory
- Content Sync: Dynamically update content from external URLs like blog posts or project descriptions, to reduce maintanence.
- Maintainable Embeds: Blog

## Style Guidelines:

- Background color: Off-white (#F0F0F0) for a clean, minimalist feel.
- Primary color: Dark gray (#333333) for text and main elements, ensuring readability and a modern look.
- Accent color: Slate blue (#708090) to highlight interactive elements and add a touch of sophistication. 
- Body font: 'Inter', sans-serif, for clean and modern body text.
- Headline font: 'Space Grotesk', sans-serif, for headings and titles, providing a tech-forward feel.
- Use simple line icons to represent skills and categories in the portfolio.
- Responsive layout adapting to different screen sizes.
