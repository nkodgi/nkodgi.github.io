# Natasha Kodgi | Portfolio

Natasha Kodgi is a Biomedical Engineer specializing in women's health, data, and human-centered design. This portfolio showcases her research, computational projects, and professional experience.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Infrastructure**: [GitHub Pages](https://pages.github.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build and export:
   ```bash
   npm run build
   ```

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions when pushing to the `main` branch.
